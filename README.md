# 🕵️‍♂️ PhantomCurl

**PhantomCurl** is a Python library for advanced web scraping that combines the raw speed and TLS impersonation of `curl_cffi` with a lightweight embedded JS engine (QuickJS + Domino). 

No Selenium. No Playwright. No external drivers. Just pure Python that executes JS, manipulates the DOM, and bypasses protections.

## 🎯 Why PhantomCurl?

- **The `requests`/`httpx` problem:** They can't execute JS. Many sites render content via React/Vue.
- **The Selenium/Playwright problem:** They are heavy, require installing browsers, eat RAM, and are easily detected.
- **The PhantomCurl solution:** A lightweight context (QuickJS) that parses HTML into a DOM tree, executes site scripts, and allows you to interact with the page, all while having a perfect TLS fingerprint of a real Chrome browser.

## ✨ Key Features

- 🚀 **TLS Impersonation:** Powered by `curl_cffi`, imitating JA3 fingerprints of Chrome/Safari. 
- 🧠 **Embedded JS Engine:** QuickJS executes JavaScript right inside the Python process memory.
- 🏗️ **Full DOM:** DOM Level 4 implementation (Mozilla Domino) for clicks, inputs, and manipulations.
- 🍪 **Storage Simulation:** Full support for Cookies, localStorage, and sessionStorage.
- 🛡️ **Anti-Detect:** Built-in injections for Canvas, WebGL, and Navigator spoofing.
- 🤖 **Captcha API:** Easy integration with 2Captcha/CapSolver.

## ⚡ Quick Start

```python
from phantom_curl import PhantomClient

with PhantomClient(impersonate="chrome110") as client:
    # Load the page (with Chrome impersonation)
    response = client.get("https://spys.one/en/anonymous-proxy-list/")
    
    # Evaluate a math expression
    print(client.evaluate("2 ** 10"))  # 1024
    
    # Get data generated by JS
    token = client.evaluate("document.getElementById('csrf-token').innerText")
    
    # Click a button
    client.click("#load-more-btn")
    
    # Read localStorage
    storage = client.get_local_storage()
